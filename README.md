# Task-2
# Basic Gates, Half Adder and Full Adder using Verilog HDL

## Project Overview
This project contains Verilog HDL implementations of:

- Basic Logic Gates
- Half Adder
- Full Adder

The project also includes testbenches and simulation waveforms for verification.

---

## Files Included

### Verilog Source Files
- `Basicgates.v`
- `halfadder.v`
- `fulladder.v`

### Testbench Files
- `tb_Basicgates.v`
- `tb_halfadder.v`
- `tb_fulladder.v`

---

## Logic Implemented

### Basic Gates
- AND Gate
- OR Gate
- NOT Gate
- NAND Gate
- NOR Gate
- XOR Gate
- XNOR Gate

### Half Adder
Inputs:
- A
- B
Outputs:
- SUM
- CARRY

### Full Adder
Inputs:
- A
- B
- CIN

Outputs:
- SUM
- COUT

---

## Simulation

The designs were simulated using Verilog testbenches.

### Verification Done
- Outputs verified with truth tables
- Correct waveform generation
- No unknown (X) states observed

- Verilog HDL
- Xilinx Vivado / Xilinx ISE
- RTL Simulation
